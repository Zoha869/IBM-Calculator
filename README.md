# 🧮 IBM Python BMI Calculator Project

## 📖 Project Description
This is a simple Python project that calculates Body Mass Index (BMI) based on user input for weight and height.  
It then classifies the BMI into different health categories.

## 🚀 Features
- Takes user input (weight & height)
- Calculates BMI using Python
- Classifies result into:
  - Underweight
  - Normal
  - Overweight
  - Obese
  - Severely Obese
  - Morbidly Obese

## 🛠️ Technologies Used
- Python
